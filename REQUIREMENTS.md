# 小学英语学习 Web App — 需求文档

## 一、项目概述

面向小学1-6年级学生的英语单词学习 Web 应用，支持单词浏览、发音、多种题型练习、闯关机制和学习进度追踪。纯前端实现，无需后端服务。

## 二、目标用户

- 小学1-6年级学生（6-12岁）
- 家长辅助使用场景

## 三、技术选型

| 项目 | 选型 |
|------|------|
| 框架 | React 18 + TypeScript |
| 构建工具 | Vite |
| 样式方案 | TailwindCSS |
| 动画 | Framer Motion |
| 单词发音 | Web Speech API（浏览器内置） |
| 数据存储 | LocalStorage（进度/成就） |
| 词库数据 | JSON 内置（无后端） |
| 路由 | React Router |
| 部署 | 静态文件，任意服务器可托管 |

## 四、内容体系

### 4.1 年级与词汇量

| 年级 | 词汇量（约） | 内容重点 |
|------|-------------|---------|
| 一年级 | 50-80词 | 字母、颜色、数字、动物、水果 |
| 二年级 | 80-120词 | 家庭、身体、食物、衣服、学校用品 |
| 三年级 | 120-180词 | 简单句型、日常对话、天气、时间 |
| 四年级 | 180-250词 | 方位、职业、日常活动、简单阅读 |
| 五年级 | 250-350词 | 自然、情感、交通工具 |
| 六年级 | 350-500词 | 学科、环境保护、未来与梦想 |

### 4.2 单元结构

- 每个年级包含 3-4 个主题单元
- 每个单元包含 8-10 个核心单词
- 每个单词包含：英文、中文释义、Emoji 配图

## 五、功能模块

### 5.1 首页 — 年级选择

- 展示 6 个年级卡片（网格布局）
- 每张卡片显示：年级名称、Emoji 图标、单元数、词汇数
- 显示该年级学习进度条（百分比）
- 顶部显示全局统计：已学单词总数、连续学习天数

### 5.2 年级页 — 单元列表

- 显示当前年级的所有单元
- 每个单元卡片包含：单元名称（中英）、单词数量、完成星级（0-3星）
- 提供两个入口：「学单词」和「闯关」
- 闯关系统：前一关需达到至少1星才能解锁下一关（第一关默认解锁）

### 5.3 单词学习页

- 卡片式单词浏览（一次一个）
- 卡片正面：Emoji + 英文单词
- 点击卡片翻转显示：中文释义
- 发音按钮：点击使用 Web Speech API 朗读英文
- 左右滑动/按钮切换上一个/下一个单词
- 进度指示器：当前第 X / 共 Y 个
- 学习完成后自动标记单词为「已学」

### 5.4 闯关练习页

#### 题目类型（4种，随机出题）

1. **看中文选英文（选择题）**
   - 显示中文释义 + Emoji
   - 4 个英文选项（1 正确 + 3 干扰项）
   - 干扰项从同单元或同年级词库中随机抽取

2. **听发音选单词（听力题）**
   - 自动播放英文发音
   - 提供重播按钮
   - 4 个英文选项供选择

3. **英文中文配对（连线题）**
   - 左侧 4 个英文单词，右侧 4 个中文释义
   - 用户依次点击左右两侧进行配对
   - 配对成功高亮，配对失败抖动提示

4. **字母拼写（填空题）**
   - 显示中文释义 + Emoji
   - 显示单词部分字母（隐藏 1-3 个字母用下划线代替）
   - 提供字母键盘供选择填入

#### 闯关规则

- 每关 10 道题（从当前单元词库中出题）
- 答对一题得 10 分，满分 100 分
- 评星标准：60分=1星，80分=2星，100分=3星
- 答错的单词自动加入「错题本」
- 答题有动画反馈（正确：绿色 + 对勾动画；错误：红色 + 抖动动画）
- 答完显示成绩单：得分、星级、答对/答错数量

### 5.5 错题本页

- 展示所有答错过的单词列表
- 每个单词显示：Emoji、英文、中文
- 点击可发音
- 提供「移除」按钮（用户认为已掌握后手动移除）
- 提供「错题重练」功能：用错题本中的单词重新生成一套练习题

### 5.6 学习统计页

- 今日学习单词数
- 累计已学单词总数
- 连续学习天数（Streak）
- 各年级完成进度（进度条）
- 各年级已获星数统计

### 5.7 成就徽章页

- 徽章墙展示所有可获得的成就
- 已解锁徽章高亮 + Emoji，未解锁灰色
- 成就列表：

| 成就ID | 名称 | 条件 |
|--------|------|------|
| first_word | 初学者 | 学会第1个单词 |
| ten_words | 小小学霸 | 学会10个单词 |
| fifty_words | 词汇达人 | 学会50个单词 |
| hundred_words | 百词斩 | 学会100个单词 |
| two_hundred | 词汇大师 | 学会200个单词 |
| streak_3 | 坚持不懈 | 连续学习3天 |
| streak_7 | 一周达人 | 连续学习7天 |
| streak_30 | 学习之星 | 连续学习30天 |
| first_unit | 闯关成功 | 完成第1个单元 |
| five_units | 过关斩将 | 完成5个单元 |
| perfect | 满分通关 | 获得一次3星通关 |

## 六、页面结构与路由

```
/                       首页（年级选择）
/grade/:gradeId         年级页（单元列表）
/learn/:gradeId/:unitId 单词学习页
/quiz/:gradeId/:unitId  闯关练习页
/wrong-book             错题本
/stats                  学习统计
/achievements           成就徽章
```

## 七、数据存储设计

所有数据存储在 LocalStorage，key 为 `english_app_data`，结构如下：

```typescript
interface ProgressData {
  completedUnits: Record<string, number>  // "年级-单元" -> 星数(1-3)
  learnedWords: string[]                  // 已学单词ID列表
  wrongWords: string[]                    // 错题单词ID列表
  dailyWords: Record<string, number>      // "日期" -> 当日学习数
  streak: number                          // 连续学习天数
  lastStudyDate: string                   // 最后学习日期
  achievements: string[]                  // 已解锁成就ID列表
}
```

## 八、UI/UX 设计要求

- **风格**：卡通、明亮、适合儿童
- **配色**：每个年级有独立主题色（红、橙、黄、绿、蓝、紫）
- **布局**：移动端优先，最大宽度 512px 居中
- **动画**：
  - 页面切换淡入动画
  - 卡片翻转动画（单词学习）
  - 答对/答错反馈动画（绿色对勾 / 红色抖动）
  - 星星飘落动画（闯关完成）
  - 成就解锁弹窗动画
- **交互**：
  - 大按钮、大文字，方便儿童点击
  - 底部固定导航栏（首页、统计、成就、错题本）

## 九、不在范围内

- ❌ 用户注册/登录系统
- ❌ 后端服务器/数据库
- ❌ 多人竞赛/排行榜
- ❌ 作文批改/AI 对话
- ❌ 视频/音频课程
- ❌ 付费/内购功能

## 十、部署方式

纯静态文件，可部署到任意静态托管服务（Nginx、Vercel、Netlify、GitHub Pages 等）。构建命令：

```bash
npm run build    # 输出到 dist/ 目录
```
